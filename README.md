## 🎯 Course Goal

Learn how to use AI in DevOps workflows to automate tasks, improve efficiency, and enhance system reliability.

---

## 👥 Who is this Course For?

- DevOps Engineers
- Cloud Engineers
- System Administrators
- Developers interested in DevOps

---

## 📚 Course Syllabus

### 🔹 Chapter 1: Introduction to AI in DevOps

- What is DevOps — brief context setting
- What is AI — simple definition
- What is AI in DevOps — combining the two
- Why AI matters in modern DevOps
- The 2 AM deployment failure story — practical example
- Where AI is used in DevOps — CI/CD, IaC, monitoring, cloud (overview)
- AI is not perfect — it's a smart assistant, not a replacement

---

### 🔹 Chapter 2: AI Fundamentals (Quick Recap)

- What is AI
- The AI family — ML, Deep Learning, Generative AI, LLMs (as a hierarchy)
- What is Machine Learning — with anomaly detection as the DevOps example
- What is Deep Learning — with log analysis as the DevOps example
- What is Generative AI — with Terraform/script generation as the DevOps example
- What are LLMs — tools like Copilot, ChatGPT, Claude explained simply
- How LLMs work — the "well-read colleague" mental model
- Practical mapping — which AI type shows up where in DevOps
- LLMs are not always right — use with engineering judgement
